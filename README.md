# Dungeon Crawler

Это 2D-игра про исследование подземелий, в которой главный герой пробивается сквозь толпы монстров, чтобы найти выход на следующий, более глубокий ярус.  
Каждый новый уровень переносит персонажа в еще более опасные залы с новыми ловушками и сильными врагами.  
Цель — зачистить все этажи подземелья.

## О проекте
Учебный проект на C#

## Основные механики, которые планируется реализовать:
- **Управление:** движение героя по уровням и боевая система.
- **Враги:** монстры, обитающие в темноте.
- **Система уровней:** проходите уровне, чтобы улучшить снаряжения героя.
- **Сбор лута:** поиск золота и улучшение снаряжения.
